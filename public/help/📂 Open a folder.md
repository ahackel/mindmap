The sidebar has two tabs: **On device** (private browser storage, works everywhere) and **Local folder**.

On **Chrome / Edge**, the Local folder tab lets you open a real folder on disk — each card becomes a `.md` file you can edit in any editor, sync, or version-control. Folders you've opened before stay in the list for one-click reopening.
