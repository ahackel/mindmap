Card notes are Markdown. Supported:

- Headings — `# Heading`
- Emphasis — `**bold**` and `*italic*`
- `inline code` and fenced code blocks
- Bullet, numbered, and `- [ ]` task lists
- Links `[text](url)` and `> quotes`

Ticking a task box updates the note on disk.
